# 環境構築

## プロジェクト初期化

npm create vite@latest . -- --template vue-ts

## 依存関係インストール

npm i

## ライブラリ導入（Three.js / Socket.io-client / 型）

npm i three socket.io-client
npm i -D @types/three
npm i -D eslint prettier eslint-config-prettier eslint-plugin-vue @vue/eslint-config-typescript
npm i vuetify
npm i -D sass vite-plugin-vuetify
